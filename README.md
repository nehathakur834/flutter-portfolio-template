# flutter-portfolio-template
A modern, responsive Flutter Web portfolio template with smooth animations and clean architecture. Easily customizable for developers to showcase projects, skills, and experience.

✨ Features
🎨 Modern & clean UI design
📱 Fully responsive (Web, Tablet, Mobile)
🌙 Light & Dark theme support
⚡ Smooth animations & transitions
🧩 Reusable widgets & clean architecture
🔗 Easy to customize content
🚀 Ready for deployment (Netlify / GitHub Pages)

🛠 Tech Stack
Flutter (Web)
Dart
Material UI
Responsive Layout
Animations (Implicit & Explicit)

📂 Project Structure
lib/
 ├── core/
 ├── data/
 ├── presentation/
 │    ├── screens/
 │    ├── widgets/
 │    ├── sections/
 ├── main.dart


 🚀 Getting Started
1. Clone the repository
git clone https://github.com/yourusername/flutter-portfolio-template.git
cd flutter-portfolio-template
2. Install dependencies
flutter pub get
3. Run the app
flutter run -d chrome
4. Build for web
flutter build web
🌐 Deployment
🔹 Netlify
Run:
flutter build web
Upload:
build/web

to Netlify

🔹 GitHub Pages
Copy:
build/web
Paste into /docs folder
Enable GitHub Pages in repo settings
